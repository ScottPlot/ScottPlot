# Contributing to ScottPlot

### ScottPlot is open source but _not_ open to code contribution at this time.

**Pull Requests:** Scott is currently [undergoing serious medical treatment](https://swharden.github.io/) 
that limits his ability to perform timely thorough code review,
so **anonymous pull requests have been temporarily disabled** starting in late March, 2026.
If you have a pull request you think is urgent, please at @ one of the other administrators in
the [ScottPlot Discord](https://scottplot.net/discord/) who will triage the topic and 
may reach out to Scott to expedite review.

**Issues:** Issues are a useful form of communication for other users, so posting new issues and
having conversations in the issues section is encouraged. When Scott returns he will review all
open issues, so topics worth addressing (e.g., topics for future PRs) are worth noting in issues now, 
and action can be taken when Scott has full availability again.

**Discord:** The [ScottPlot Discord](https://scottplot.net/discord/) is an active community with over 1000 members! 

**First time Contributors:**  See http://scottplot.net/contributing/
